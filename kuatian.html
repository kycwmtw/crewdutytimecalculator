✈️ 航班跨日檢查工具

上傳檔案，系統會自動篩選 TPE 起訖、74Y / 77X 的航班， 檢查 STD/STA 與 ETD/ETA/ATD/ATA 是否跨日，並輸出最終清單。

📋 使用說明

  1. 使用 WebJC，選取日期時間區間
  2. BY 選 STD(LOCAL)、STA(LOCAL)
  3. TIME_TYPE 選 LOCAL
  4. 按 Query
  5. 按 Export Excel
  6. 將產出的檔案直接拖入本系統即可分析
⚠️ 注意：最好是 STD(LOCAL)、STA(LOCAL)兩種條件都各 Query 一次，並分別產出 Excel 檔，兩個檔都拖進來分析！（因為兩種條件下包含的航班可能有不一樣）
